backup

Tempo real

Dados online e em tempo real. As estações coletam os dados e enviam automaticamente. Os dados ficam disponíveis aqui e no seu site também!

lock_open

Open source

Todo o código fonte das estações é aberto. Hardware e firmware estão disponíveis no repositório do GitHub e o envio de dados e uso do site são gratuitos!

settings

Tenha uma também!

Você quer uma estação meteorológica na sua casa? É fácil montar a sua, ou se preferir, você pode comprar uma!

Gráficos e informações

Cada estação tem uma página onde são exibidos os dados atuais e também, os dados das últimas horas em forma de gráfico, que facilitam muito a leitura e interpretação das informações

Monte a sua

Uma estação Pluviam pode ser montada com qualquer hardware IoT (Internet of Things), Arduino, ThinkIt e outros. No repositório do GitHub, todas as intruções são claras e fáceis de seguir, mas se surgir alguma dúvida, você pode entrar em contato pelas redes sociais, formulário de contato ou também diretamente no email hello@pluvi.am. Se você preferir, também pode comprar a estação montada e configurada, pronta para instalar!

Sobre a Pluviam

A pluviam é uma rede de estações meteorológicas automáticas. Qualquer pessoa, empresa ou instituição pode montar uma e enviar os dados para o site. O envio e armazenamento dos dados é totalmente gratuito, assim como o acesso as informações de todas as estações.
